The Untermyer Garden Conservancy Project

Problem

Users feel that the Untermyer Gardens Conservancy website is outdated and cluttered, especially in the header section, which makes it difficult to navigate and find relevant information, highlighting a need for a modern, streamlined, and user-friendly redesign.

Solution

By Restructuring the Navigation Bar, we figured it would have a high impact on helping lessen confusion for users when trying to find a certain page

Additionally, we decided to increase font size as it would help increase accessibility and improve readability

User Flow

We prioritized these features after conducting our stakeholder interview and our user interviews. We wanted to make the process of learning about and setting up tours much easier
